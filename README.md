# نظام إدارة المرضى - Clinic System

هذا مشروع ويب يهدف لإدارة مراجعات المرضى والأطباء في المستشفى أو العيادة باستخدام PHP و MySQL.

## أهداف المشروع
- تسجيل دخول الموظف المسؤول
- إضافة المرضى وتسجيل مراجعاتهم
- إضافة الأطباء
- البحث والفلترة حسب الاسم أو القسم
- تسجيل خروج المرضى وتحديث بيانات الدخول والخروج

## التكنولوجيا المستخدمة
- لغة PHP للباك إند
- قاعدة بيانات MySQL لتخزين البيانات
- HTML و CSS للواجهة

## بنية المشروع
- `login.php` : صفحة تسجيل دخول الموظف
- `dashboard.php` : لوحة التحكم وعرض المرضى
- `add_patient.php` : إضافة مريض جديد
- `add_doctor.php` : إضافة طبيب جديد
- `doctors.php` : عرض قائمة الأطباء
- `logout.php` : تسجيل خروج الموظف
- `db/config.php` : الاتصال بقاعدة البيانات
- `css/style.css` : تنسيق واجهة الموقع
- `sql/database.sql` : قاعدة البيانات والجداول

## خطوات تشغيل المشروع محليًا
1. تثبيت XAMPP وتشغيل:
   - Apache
   - MySQL
2. نسخ مجلد المشروع إلى:

C:\xampp\htdocs\clinic-system

3. استيراد قاعدة البيانات:
- فتح `phpMyAdmin`
- اختيار قاعدة البيانات `clinic_system`
- استيراد ملف `sql/database.sql`
4. فتح المتصفح والذهاب إلى:
http://localhost/clinic-system/login.php

5. بيانات تسجيل الدخول الافتراضية:
- اسم المستخدم: `admin`
- كلمة المرور: `1234`

## ملاحظات مهمة
- تأكد أن جميع الملفات لها امتداد صحيح:
- `.php` للصفحات  
- `.css` للتنسيق  
- `.sql` لقاعدة البيانات
- XAMPP يجب أن يكون Apache و MySQL شغالين قبل فتح الموقع
- البيانات المخزنة في قاعدة البيانات تبقى محفوظة حتى بعد إغلاق الجهاز
